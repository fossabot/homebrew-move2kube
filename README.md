# Move2Kube - Homebrew

Brew formula for [Move2Kube](https://github.com/konveyor/move2kube)

## Installation
```
brew tap konveyor/move2kube
brew install move2kube
```

After installation run
```
move2kube version
```
to make sure it was installed properly.
